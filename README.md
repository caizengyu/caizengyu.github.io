# 个人主页内容

## 个人简介
蔡增玉，男，1979年12月生，教授，硕士生导师，主要从网络空间安全、人工智能、信息系统开发等方面的研究工作。中国计算机学会会员、中国人工智能学会会员、ACSIT会员。全国大学生信息安全竞赛评审专家、全国研究生教育评估监测专家库专家、全国本科毕业论文（设计）抽检评审专家库。

## 工作经历
2008年至今，郑州轻工业大学计算机与人工智能学院，教授
2003-2006年，东北师范大学计算机应用技术专业，研究生
1999-2003年，东北师范大学计算机科学与技术专业，本科

## 研究方向
网络空间安全、人工智能、信息系统开发

## 招生简介
每年招收硕士生 4-6 名。鼓励学生参与真实课题，支持国际会议交流与联合培养。

## 招生要点
- 招收对象：计算机、网络安全等相关专业。
- 建议准备：数学基础（线代/概率）、Python 编程、英文阅读能力。
- 申请方式：发送简历、成绩单和研究兴趣说明到联系邮箱。
